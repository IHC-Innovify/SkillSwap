                                                                    SkillSwap - Landing Page (Página Web)

Este repositorio contiene el código fuente (HTML y CSS) de la página web principal (Landing Page) para el proyecto SkillSwap (también conocido como Innovify).

SkillSwap es una plataforma en línea diseñada para conectar a estudiantes de distintas universidades, permitiéndoles compartir e intercambiar conocimientos bajo un modelo de trueque.

 Estructura del Repositorio
